# IoT Tools

# Basic install

```bash
$> curl -sL https://raw.githubusercontent.com/vforge/iott/master/scripts/setup.sh | bash
```

# Installing node

```bash
$> curl -sL https://deb.nodesource.com/setup_11.x | sudo -E bash -
```

```bash
$> sudo apt install -y nodejs
```

# Renaming pi

```bash
$> sudo nano /etc/hostname
```
